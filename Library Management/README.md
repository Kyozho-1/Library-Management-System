Chase's Library Management Application
Version: 1.0.0

Features
--------
- Add users
- Remove users
- Add books
- Remove books
- Borrow books
- Return books
- Search books
- View all users
- View all books

Requirements
------------
- Windows
- Visual Studio 2022
- C++20

Known Limitations
-----------------
- Data is not saved between sessions
- No due dates or fines
- No duplicate ISBN protection

Future Plans
------------
- Save/load data
- Improved search
- Better validation
- Additional book management features
